# Priceit alpha

Priceit alpha is the early public-facing shell for a Canadian real estate pricing product.

## Current direction

- Phase 1 established the product concept with a live alpha experience.
- Phase 2 focuses on moving from city-level estimates toward neighborhood-level pricing accuracy.
- Future phases aim to improve micro-market precision and richer pricing intelligence.
